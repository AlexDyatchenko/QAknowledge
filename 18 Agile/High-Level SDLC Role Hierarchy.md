# 2. Role Breakdown (With Headcount, Responsibility, Interaction)

## 1. Executive Sponsor (1)

**Responsibility**

* Owns business outcome & funding
* Approves roadmap, major scope changes
* Removes organisational blockers

**Interacts With**

* Product Manager
* Delivery Manager
* Stakeholders

---

## 2. Product Manager (1)

**Responsibility**

* Owns product vision & roadmap
* Prioritises backlog
* Defines success metrics

**Key Deliverables**

* Product roadmap
* OKRs
* Feature prioritisation

**Interacts With**

* Sponsor
* Business Analyst
* UX
* Tech Lead
* QA Lead

---

## 3. Delivery Manager / Scrum Master (1)

**Responsibility**

* Ensures delivery cadence
* Facilitates Agile ceremonies
* Removes delivery impediments

**Key Deliverables**

* Sprint planning
* Release coordination
* Velocity & delivery metrics

**Interacts With**

* All team members
* Product Manager
* Tech Lead

---

## 4. Solution Architect (1)

**Responsibility**

* Defines system architecture
* Ensures scalability & integration patterns
* Aligns with enterprise standards

**Key Deliverables**

* Architecture diagrams
* Tech decisions (ADRs)

**Interacts With**

* Tech Leads
* DevOps
* Security
* Product Manager

---

## 5. Technical Lead(s) (2)

**Responsibility**

* Own code quality & technical execution
* Mentors engineers
* Approves technical design

**Key Deliverables**

* Design reviews
* Code standards
* Technical debt management

**Interacts With**

* Developers
* QA Lead
* DevOps
* Architect

---

## 6. Backend Engineers (5)

**Responsibility**

* API development
* Business logic
* Database integration

**Key Deliverables**

* APIs
* Microservices
* Unit tests

**Interacts With**

* Frontend
* QA
* DevOps

---

## 7. Frontend Engineers (4)

**Responsibility**

* Web UI development
* UX implementation
* Accessibility & performance

**Key Deliverables**

* UI components
* Frontend tests

**Interacts With**

* UX Designer
* Backend
* QA

---

## 8. Mobile Engineers (2) *(Optional)*

**Responsibility**

* Native or cross-platform apps
* App store releases

**Interacts With**

* Backend
* QA
* UX

---

## 9. QA Lead (1)

**Responsibility**

* Defines test strategy
* Owns quality gates
* Coordinates test efforts across SDLC

**Key Deliverables**

* Test strategy
* Release sign-off
* Quality metrics

**Interacts With**

* Product Manager
* Tech Lead
* Automation & Manual QA
* DevOps

---

## 10. Automation QA Engineers (3)

**Responsibility**

* API/UI automation
* CI integration
* Test data & mocking (WireMock, etc.)

**Key Deliverables**

* Regression suites
* CI pipelines test stages

**Interacts With**

* Developers
* DevOps
* QA Lead

---

## 11. Manual QA Engineers (2)

**Responsibility**

* Exploratory testing

**Interacts With**
